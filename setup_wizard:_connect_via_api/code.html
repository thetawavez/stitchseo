Setup

Connect your WordPress site

Install our plugin and enter your API key to sync your site and start getting SEO insights.

1

Install the plugin

In your WordPress dashboard, go to Plugins > Add New, search for "SEO AI", and install it.

2

Get your API key

Find your API key in your account settings on our web app under the 'API Keys' section.

3

Enter your API key

Paste the key into the settings page of the "SEO AI" plugin in your WordPress admin area.
